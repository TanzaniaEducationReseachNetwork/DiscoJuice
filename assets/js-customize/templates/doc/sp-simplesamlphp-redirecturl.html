Finding a redirectURL for SimpleSAMLphp

May be inside your application, you have a login page that initates login, then it would be appropriate to redirect to this page.

SimpleSAMLphp also has a built in URL that initates login. 

https://{{ep_host}}/simplesaml/module.php/core/as_login.php?
	AuthId=saml&ReturnTo=[URL]&idp=[IdP]

If you want to redirect back to the current page, you may do something like this:

// Change if you use another authentication source for SAML
var authenticationSource = 'saml'; 
var redirectURL = 'https://{{ep_host}}/simplesaml/module.php/core/as_login.php?' +
	'AuthID=' + authenticationSource + '&' +
	'ReturnTo=' + encodeURIComponent(window.location.href) +
	'&idp=';

DiscoJuice.Hosted.setup(
	{
		....
		"redirectURL": redirectURL,
		....
	}
);
