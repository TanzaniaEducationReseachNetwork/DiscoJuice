Setting up SimpleSAMLphp with an IdP Discovery Service

In the config/authsources.php configuration file, configure the SAML authenciation source with the correct discovery endpoint (discoURL).

	/*
	 * This is the default SAML SP, used in examples.
	 */
	'saml' => array(
		'saml:SP',

		/*
		 * The entity ID of the IdP this should SP should contact.
		 * Can be NULL/unset, in which case the user will be shown a list of available IdPs.
		 */
		'idp' => NULL,
		'discoURL' => '{{globalURL}}',
	),
